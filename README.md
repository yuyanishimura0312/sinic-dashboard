# SINIC経営ダッシュボード

SINIC循環経営モデル（SCM）に基づくバックキャスティング経営管理ツール。

## 概要

SINIC経営コンソーシアム2025の最終レポートで提案された「SINIC循環経営モデル」を実装するWebダッシュボード。従来の財務KPIを補完する「社会段階移行への準備度」を測定・可視化し、組織の未来への応答力を高める。

## 主な機能

- **KPIダッシュボード**: 4カテゴリの評価指標（社会段階知覚/段階移行準備度/循環活性度/意味生成活性度）
- **社会段階タイムライン**: 最適化社会→自律社会→自然社会の移行シグナルをリアルタイムモニタリング
- **バックキャスティング・ポートフォリオ**: Probe（事業実験）の登録・評価・管理
- **未来への応答（Correspondence）**: 「避け難い未来」からの問いかけに対する組織的応答の蓄積
- **五循環モニター**: Seed-Need/段階移行/翻訳/共創/身体知の5循環の活性度測定
- **シグナルスキャン**: 社会段階移行シグナル（Seed型/Need型）の検知・記録

## セットアップ

```bash
# ローカルで開く
open index.html

# または簡易サーバー
python3 -m http.server 5200
```

## 技術

- Vanilla JavaScript（フレームワーク不使用）
- CSS Grid/Flexbox
- モックアップ段階（データはハードコード）
